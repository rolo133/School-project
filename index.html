input, output en verwerking van informatie en gegevens

de input van een computer is alles wat een computer van buiten binnenkrijgt.
Dit kan verschillende dingen zijn zoals een toetsenboord of muis.
De imput wordt van een fysieke beweging of aanraking veranderd in een signaal
dat een computer kan begrijpen. Nadat deze informatie binnen is gekomen
bij de computer wordt het verwerkt. De CPU voert dan instructies uit die opgeslagen
zijn in het geheugen. Als de informatie dan verwerkt is wordt de
informatie opgeslagen in de RAM. dit zoorgt ervoor dat de informatie
snel weer hergebruikt kan worden door de computer als het nodig is.
Als de informatie langer moet worden opgeslagen wordt het opgeslagen op
de harde schijf. Nadat de informatie compleet verwerkt is maakt de
computer een output die hij aan ons mensen kan laten zien.
Als je bijvoorbeeld dus een letter intikt op je toetsenboord
is de input de knop, daarna zal de computer een signaal krijgen
waardoor het weet wat het moet doen en dan stuurt het een signaal
terug dat verandert in een letter die wij kunnen lezen.
