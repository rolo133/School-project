wat is de toepassingenlaag

De toepassingenlaag is de laag waar de toepassing op staat waarvoor
de twee andere lagen samen werken zodat het kan werken. Hiertoe behoren
alle applicaties die op een computer staan.

Apps

Apps of applicaties zijn programma's die bedoelt zijn voor de eindgebruiker
anders dan hardware of software runnen apps op de achtergrond. Apps runnen
op of onder een bepaalde OS. Sommige apps kunnen worden gebruikt op verschillende
OS en zijn daarom multiplatform. Elke app heeft een andere complexiteit het kan
uitlopen van simpel zoals games en tekseditors naar extreem complex zoals
luchtverkeers apps of digitaal bankieren.

API's

Een API of application programming interface is een programma meestal in de
vorm van een bibliotheek die ervoor zorgt dat een computerprogramma kan
communiceren met een ander programma. Een API zoorgt er bijvoorbeeld voor
dat een programma dat goed met abstractie kan werken veel meer met abstractie
werkt en het niet abstracte werk overlaat aan een ander programma dat dat veel
minder goed kan. Een voorbeeld hiervan is een tekenprogramma en een printer.
Het tekenprogramma weet niet hoe het een printer moet aansturen dus zorgt een
API ervoor dat het tekenprogramma dat ook niet hoeft te doen en zet een ander
programma hiervoor aan.
