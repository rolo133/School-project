Wat is de fysieke laag

De fysieke laag is de laag met alle hardwear van de computer zoals het ROM en RAM geheugen,
de processor en de CPU. Op deze pagina zal ik meer vertellen over de onderdelen van de fysieke laag.

het geheugen

Het geheugen van een computer bestaat uit drie onderdelen: de ROM, de RAM en het permanent geheugen.
Het RAM of random-acces memory is het stuk geheugen dat de code bevat die je de computer geeft. Als je
meer RAM geheugen hebt zal de computer ook sneller de opdrachten kunnen uitvoeren die je hebt gegeven.
Het ROM of Read only memory is het stuk geheugen waarop de firmware van de computer staat. Het ROM wordt
gebruikt wanneer een computer een programma moet opslaan of als een programma bijna nooit hoeft worden
aangepast. Het ROM is significant langzamer dan het RAM
Het permanent geheugen is het stuk geheugen dat de gegevens van de computer kan bewaren ook al staat deze
uit. Meestal wordt het permanent geheugen gebruikt als lange termijn geheugen.

processor

De procesoor ook wel de CPU of Central processing unit genoemd is een stuk hardware van de computer dat de code
voor de computer controleert en uitvoert. De werking van een CPU bestaat uit vier fases: de ophaalfase, decodeerfase,
uitvoefase en opslagfase.
De ophaalfase is de fase waarin de CPU de code uit het RAM geheugen haalt en daarna wordt het in het instructieregister
geplaatst.
De decodeerfase is de fase waarin de CPU de opdracht die bestaat uit bits uitelkaar gehaald en er worden van de bits
getallen gemaakt waarmee de computer weet wat het moet doen.
De uitvoerfase is de fase waarin de CPU de getallen die zijn gemaakt bij de decodeerfase bij elkaar optelt en in een
tijdelijk stukje geheugen opgeslagen.
De opslag fase is de fase waarin de CPU luistert naar de leidingen. Als er een getal langs komt dan zal dat getal op
de plek komen van het berekende getal en het berekende getal gaat terug in de leidingen.
De snelheid van een CPU ligt aan de kloksnelheid. De kloksnelheid is de meting van hoeveel signalen de computer per
seconde afleest. Dit wordt aangeduid in hertz. Bijvoorbeeld 1Hz is dat een computer één signaal kan lezen per seconde.
