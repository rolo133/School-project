Een bitmap

Een bitmap is een online afbeelding waarin alle pixels een eigen kleur krijgen.
Je kan een bitmap eigenlijk zien als een raster met pixels.
Dit zie je ook als je inzoomt op een bitmap afbeelding.
Dan zie je dat de randen blokjes zijn en niet een mooie lijn. De grootte van
een bitmap is de resolutie van de afbeelding. Een voorbeeld van een bitmap
is een jpg.

Een vectorafbeelding

Een vectorafbeelding is een afbeelding die in tegenstelling tot een bitmap niet
van pixels is gemaakt, maar van punten. De punten worden verbonden door lijnen
waardoor er een afbeelding ontstaat. Een bitmap gebruikt een rooster terwijl een
vectorafbeelding instructies zijn die je aan de computer geeft over waar elk punt
moet liggen. Hierdoor zie je geen verschil in kwaliteit als je inzoomt op een vectorafbeelding.
Een voorbeeld van een vectorafbeelding is een svg. Op deze pagina kan je een voorbeeld zien van
een vectorafbeelding in de vorm van een tijger. Hier onder staat hetzelfde plaatje in bitmap.
Als je inzoomt op beide plaatjes kun je goed het verschil zien: blokkerig bij de bitmap en scherp bij de vectorafbeelding.
