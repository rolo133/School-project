een bitmap

een bitmap is een online afbeelding waarin alle pixels een eigen kleur krijgen.
Je kan een bitmap eigenlijk zien als een raster met pixels die allemaal een
eigen kleur krijgen. Dit zie je ook als je inzoemt op een bitmap afbeelding
dan zie je dat de rande blokjes zijn en niet een mooie lijn. De grote van
een bitmap is de resolutie van de afbeelding. Een voorbeeld van een bitmap
is een jpg.

een vectorafbeelding

een vectorafbeelding is een afbeelding die integenstelling tot een bitmap niet
van pixels is gemaakt, maar van punten. De punten worden verbonden door lijnen
waardoor er een afbeelding ontstaat. Een bitmap gebruikt een rooster terwijl een
vectorafbeelding instructies zijn die je aan de computer geeft over waar elk punt
moet liggen. Hierdoor zie je geen verschil in kwaliteit als je inzoemt op een vectorafbeelding.
een voorbeeld van een vectorafbeelding is een svg. Op deze pagina kan je een voorbeeld zien van
een vectorafbeelding in de vorm van een tijger. Als je inzoemt kan je het verschil zien tussen
dit plaatje en de afbeelding op deze pagina van dezelfde tijger maar dan in een bitmap.
