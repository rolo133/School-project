RGB model

Het RGB model is een model van de kleuren rood,groen en blauw.
Met dit model kan je alle kleuren maken. Alle kleuren krijgen
een eigen code bestaande uit de benodigde basiskleuren (RGB).
De code bestaat uit een getal dat vaak uit 8
bits bestaat. De getallen voor de basiskleuren kunnen variÃ«ren tussen 0 en 255,
afhankelijk van de tint van de basiskleur (bijvoorbeeld donkerblauw of lichtblauw)

CMY-K model

Het CMY-K model wordt gebruikt voor het maken van veel verschillende
kleuren door subtractieve kleurmenging. Het wordt vooral gebruikt voor
drukinkten. CMY-K staat voor cyan, magenta, yellow en key en betekent dat
deze kleuren voor alle kleuren zorgen die je ziet. Soms wordt zwart
weggelaten, maar dit wordt weinig gebruikt omdat afbeeldingen dan te licht
of bruinachtig uitgeprint worden.

Pixels en resolutie

Pixels zijn de kleurpunten in een computer. Als er veel bij elkaar
komen te staan vormt het een beeld en dus alles wat je op een digitaal
beeld ziet is opgebouwd met pixels. De resolutie van een afbeelding is hoeveel
pixels er in de afbeelding aanwezig zijn. Elke pixel heeft drie kleuren,
rood, groen en blauw. De kleuren bij elkaar maken andere kleuren en al de
verschillende kleuren bij elkaar worden dan een plaatje.
