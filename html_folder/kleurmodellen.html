RGB model

het RGB model is een model van de kleuren Rood,Groen en blauw.
met dit model kan je alle kleuren maken. Alle kleuren krijgen
een eigen code bestaande uit de benodigde aantal kleuren van
de basiskleuren. De code bestaat uit een getal dat vaak uit 8

CMY-K model

Het CMY-K model wordt gebruikt voor het maken van veel verschillende
kleuren door subtractieve kleurmenging. Het wordt vooral gebruikt voor
drukinkten. CMY-K staat voor Cyaan, maganta, yellow, key en betekend dat
deze kleuren voor alle kleuren zorgen die je ziet. Soms wordt zwart
weggelaten, maar dit wordt weinig gebruikt omdat afbeeldingen dan te licht
of bruinachtig uitgeprint worden.

pixels en resolutie

Pixels zijn de kleurpunten in een computer. Als er veel bijelkaar
komen te staan vormd het een beeld en dus alles wat je op een digitaal
beeld ziet is gemaakt van pixels. De resolutie van een afbeelding is hoeveel
pixels er in de afbeelding aanwezig zijn. Elke pixel heeft drie kleuren,
rood, groen en blauw. De kleuren bijelkaar maken andere kleuren en al de
verschillende kleuren bijelkaar worden dan een plaatje.
